## 常见问题

1. **是否支持奈飞和迪士尼？**

   除香港节点未解锁版权剧外，其余节点均支持解锁Netflix和Disney+，但不保证100%稳定性。精品线路提供更多节点，但也无法保证全部流媒体的稳定解锁。

2. **SLA是什么意思？**

   * **高可用性**：高带宽可用性，确保连接稳定
   * **高优先级**：对于修复和优化的优先级较高
   * **高优先售后保障**：优先处理用户的反馈和问题

3. **不同等级套餐有什么区别？**

   * **限速**：更高等级套餐提供更高带宽速度
   * **节点**：高级套餐提供更多地区的节点
   * **稳定性**：高级套餐提供更高的稳定性保障

4. **WgetCloud提供哪些节点？**

   目前提供的常用节点包括：香港、日本、台湾、美国、新加坡，每个地区至少有三个节点。精品套餐还提供更多地区的额外节点。
